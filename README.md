# AI Resume & Job Match Assistant

An AI-powered web app that compares a resume with a job description and provides an ATS-style score, matched skills, missing skills, and resume improvement suggestions.

## Tech Stack

- Python
- FastAPI
- NLP
- HTML/CSS/JavaScript
- Docker

## Features

- Upload resume
- Paste job description
- Extract resume skills
- Extract job keywords
- Calculate match score
- Suggest resume improvements
